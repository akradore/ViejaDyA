Export Products Images

Export the bunch of products images in .zip file.

Export Images

Select the list of products and click on action bar. We can see the Export Image option menu that's highlighted in the image. When we click on menu pop up the one form view wizard.

In form view wizard we need to do some configuration.

  • Zip Name: Name of the zip file that will we export.
  • Image Size: Size of the image that you want.

And at the last, List of products whose image will export and click on Export button.

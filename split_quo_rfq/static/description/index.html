Split or Extract Quotation, Sale Order, RFQ, Purchase Order

Split Quotation Useful to perform below functions:
Split function - split function helpful to split selected order lines and create new quotation and remove selected lines from existing order.
Extract function - Extract helpful to extract sale order lines without removing from existing order.

Split RFQ Useful to perform below functions:
Split function - split function helpful to split selected order lines and create new RFQ and remove selected lines from existing order.
Extract function - Extract helpful to extract purchase order lines without removing from existing order.
